* flash

=flash= lets you navigate your code with search labels, enhanced character motions, and Treesitter integration.

Inspired by [[https://github.com/folke/flash.nvim][flash.nvim]] for Neovim.

** Features

- *Search Integration*: integrate =flash= with your regular search using ~/~ or =C-s=. Labels appear next to the matches, allowing you to quickly jump to any location. Labels are guaranteed not to exist as a continuation of the search pattern.
- *Type as many characters as you want* before using a jump label.
- *Enhanced =f=, =t=, =F=, =T= motions* with optional jump labels.
- *Treesitter Integration*: all parents of the Treesitter node under your cursor are highlighted with a label for quick selection of a specific Treesitter node.
- *Jump Mode*: a standalone jumping mode similar to search.
- *Multi Window* jumping.
- *Evil integration*: works seamlessly with evil-mode.

** Requirements

- Emacs >= *27.1*
- Emacs >= *29.1* for Treesitter support
- Optional: [[https://github.com/emacs-evil/evil][evil-mode]] for Vim-style keybindings

** Installation

*** use-package (from local path)

#+begin_src emacs-lisp
(use-package flash
  :load-path "~/path/to/flash"
  :commands (flash-jump flash-evil-jump
             flash-treesitter flash-treesitter-evil)
  :bind ("s-j" . flash-jump)
  :custom
  (flash-multi-window t)
  :init
  ;; Evil integration
  (with-eval-after-load 'evil
    (require 'flash-evil)
    (evil-global-set-key 'normal (kbd "s") #'flash-evil-jump)
    ;; Char motions (f/t/F/T)
    (flash-char-setup-evil-keys))
  :config
  ;; Search integration (flash labels during / and ?)
  (require 'flash-isearch)
  (flash-isearch-mode 1))
#+end_src

*** straight.el

#+begin_src emacs-lisp
(straight-use-package
 '(flash :type git :host github :repo "chestnykh/flash"))
#+end_src

** Configuration

=flash= is highly configurable.

*** Basic Options

| Variable                        | Default                        | Description                               |
|---------------------------------+--------------------------------+-------------------------------------------|
| ~flash-labels~            | ~"asdfjkl;ghqwertyuiopzxcvbnm"~ | Characters for jump labels                |
| ~flash-label-uppercase~   | ~nil~                          | Also use uppercase labels (A, S, D...)    |
| ~flash-multi-window~      | ~t~                            | Search in all visible windows             |
| ~flash-autojump~          | ~nil~                          | Auto-jump when single match               |
| ~flash-backdrop~          | ~t~                            | Dim non-matching text                     |
| ~flash-case-fold~         | ~t~                            | Case-insensitive search                   |
| ~flash-rainbow~           | ~nil~                          | Use rainbow colors for labels             |
| ~flash-highlight-matches~ | ~t~                            | Highlight matched text                    |
| ~flash-label-position~    | ~'after~                       | Label position: after, before, overlay, eol |
| ~flash-jump-position~     | ~'start~                       | Cursor position after jump: start, end    |
| ~flash-jumplist~          | ~t~                            | Save position to mark ring                |
| ~flash-multi-char-labels~ | ~nil~                          | Use multi-char labels (aa, as, ad...)     |
| ~flash-min-pattern-length~ | ~0~                           | Min pattern length to show labels         |

*** Char Motion Options

| Variable                          | Default | Description                     |
|-----------------------------------+---------+---------------------------------|
| ~flash-char-jump-labels~    | ~nil~   | Show labels for f/t/F/T matches |
| ~flash-char-multi-line~     | ~nil~   | Search beyond current line      |
| ~flash-char-reserved-labels~ | ~""~   | Chars to exclude from labels    |

*** Search Integration Options

| Variable                         | Default | Description                    |
|----------------------------------+---------+--------------------------------|
| ~flash-isearch-enabled~    | ~t~     | Show labels during search      |
| ~flash-isearch-toggle-key~ | ~"C-;"~ | Key to toggle labels in search |
| ~flash-isearch-trigger~    | ~nil~   | Trigger char before label jump |

*** Example Configuration

#+begin_src emacs-lisp
(use-package flash
  :custom
  (flash-labels "asdfjkl;ghqwertyuiopzxcvbnm")
  (flash-label-uppercase t)  ; double available labels (a-z + A-Z)
  (flash-multi-window t)
  (flash-autojump t)          ; optional: auto-jump on single match
  (flash-backdrop nil)
  (flash-rainbow t)
  (flash-highlight-matches t)
  (flash-label-position 'after)
  (flash-char-jump-labels t)
  (flash-nohlsearch t))
#+end_src

** Usage

*** Standalone Jump

#+begin_src emacs-lisp
M-x flash-jump
#+end_src

Or with evil: press =s= (if configured).

Type characters to narrow matches, then press a label to jump.

*** Search Integration

With =flash-isearch-mode= enabled:
- Use ~/~ or =?= (evil) or =C-s= (isearch)
- Labels appear on matches
- Press label to jump directly
- =C-;= to toggle labels on/off

*** Character Motions (f/t/F/T)

After =flash-char-setup-evil-keys=:
- =f{char}= - jump to char forward
- =t{char}= - jump to before char forward
- =F{char}= - jump to char backward
- =T{char}= - jump to after char backward
- =;= - repeat motion
- =,= - repeat motion reverse

With =flash-char-jump-labels= enabled, labels appear on multiple matches.

*** Treesitter

#+begin_src emacs-lisp
M-x flash-treesitter       ; select node with labels
M-x flash-treesitter-evil  ; same with evil visual mode
#+end_src

Shows labels for nested syntax nodes at point. Press label to select entire node.

*** Customizing Faces

#+begin_src emacs-lisp
;; Change label appearance
(set-face-attribute 'flash-label nil
                    :background "#ff0000"
                    :foreground "#ffffff"
                    :weight 'bold)

;; Change match highlight
(set-face-attribute 'flash-match nil
                    :underline nil
                    :background "#ffff00"
                    :foreground "#000000")
#+end_src
